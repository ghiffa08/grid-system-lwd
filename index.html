Teori Dasar CSS Grid

Apa itu CSS Grid?

CSS Grid adalah sistem layout dua dimensi yang memungkinkan kita untuk membuat layout kompleks dengan lebih mudah dan fleksibel. Tidak seperti Flexbox yang hanya berfokus pada satu dimensi (baris atau kolom), Grid dapat menangani baris dan kolom secara bersamaan.

Kelebihan CSS Grid:

  • Layout dua dimensi (baris dan kolom)
  • Kontrol yang lebih presisi
  • Dapat menangani layout yang kompleks
  • Mempermudah penataan item
  • Fleksibel dan responsif

Kasus Penggunaan:

  • Layout website secara keseluruhan
  • Galeri foto
  • Dashboard
  • Formulir kompleks
  • Tata letak kartu/card

Konsep Dasar

1. Container dan Item

Grid Layout terdiri dari dua komponen utama:

  • Grid Container: Elemen induk yang menjadi wadah grid
  • Grid Item: Elemen anak langsung dari grid container 
.container {
  display: grid;  /* Menjadikan elemen sebagai grid container */
}

/* Semua elemen anak langsung dari container akan menjadi grid item */

2. Grid Lines, Tracks, dan Areas

Grid Lines

Garis horizontal dan vertikal yang membentuk struktur grid.

Grid Tracks

Ruang antara dua grid lines, berupa baris (row) atau kolom (column).

Grid Areas

Area yang dibatasi oleh empat grid lines, dapat diisi oleh grid item.

Properti Penting CSS Grid

Properti Deskripsi Contoh
display: grid Mendefinisikan elemen sebagai grid container display: grid;
grid-template-columns Menentukan jumlah dan ukuran kolom dalam grid grid-template-columns: 1fr 2fr 1fr;
grid-template-rows Menentukan jumlah dan ukuran baris dalam grid grid-template-rows: 100px auto;
gap Menentukan jarak antara baris dan kolom gap: 10px;
grid-column Menentukan posisi item dalam kolom grid-column: 1 / 3;
grid-row Menentukan posisi item dalam baris grid-row: 2 / span 2;
grid-area Shorthand untuk grid-row dan grid-column grid-area: 1 / 1 / 3 / 3;
justify-items Menyelaraskan item secara horizontal dalam cell justify-items: center;
align-items Menyelaraskan item secara vertikal dalam cell align-items: center;

Unit Pengukuran dalam Grid

Fraction Unit (fr)

Unit fr membagi ruang yang tersedia secara proporsional. 

grid-template-columns: 1fr 2fr 1fr;
/* Kolom kedua akan 2x lebih lebar dari kolom pertama dan ketiga */

Fungsi repeat()

Membuat pola berulang dengan lebih mudah.

grid-template-columns: repeat(3, 1fr);
/* Sama dengan: 1fr 1fr 1fr */

Contoh Kode Lengkap

HTML

<div class="grid-container">
  <div class="grid-item">1</div>
  <div class="grid-item">2</div>
  <div class="grid-item">3</div>
  <div class="grid-item">4</div>
  <div class="grid-item">5</div>
  <div class="grid-item">6</div>
</div>

CSS

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: auto auto;
  gap: 10px;
  padding: 10px;
}

.grid-item {
  background-color: #5DBCD2;
  color: white;
  padding: 20px;
  text-align: center;
}

/* Item khusus */
.grid-item:nth-child(1) {
  grid-column: 1 / 3; /* Span dari kolom 1 sampai 3 */
}
